Dataset types

Data Link Dataset Type
UFO link Table - Contains items and attributes
Airports link Geometry - Contains items and positions
Google-web link Network - Contains nodes and links

Data Types

UFO

  • Row - Item (Table structure)
  • Datetime - Attribute
    • Quantitative,sequantial,cyclic - Time is cyclic and years are sequential, so best to treat as both.
  • City - Attribute
    • Categorical - Qualitative with discrete answers.
  • State - Attribute
    • Categorical - Qualitative with discrete answers.
  • Country - Attribute
    • Categorical - Qualitative with discrete answers.
  • Shape - Attribute
    • Categorical - Qualitative with discrete answers.
  • Duration - Attribute
    • Quantitative,sequential. - Measured in seconds, which can only increase.

Airports

  • Row - Item (Table structure)
  • Airport Name - Attribute
    • Categorical - Qualitative with discrete answers.
  • Latitude - Position (as it is a coordinate)
  • Longitude - Position (as it is a coordinate)

Google-web

  • Row - Link (each row acts as a link from one node id to another)
  • FromNodeId - Node (id of a node)
  • ToNodeId - Node (id of a node)
